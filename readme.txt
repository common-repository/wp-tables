== Description ==

- Easy way to display your wordpress posts data in HTML Tables.
- Display your wordpress pages, posts or custom post type including your custom meta field in table with pagination, filtering sorting and searching.
- you can configure post,pages orderby and per page for table.
- Arrange and manage table column order with just drag & drop feature.
- Edit table header column text and use custom text.

== Installation ==
1. Simply install and activate WP Tables plugin.
2. Now you can see WP Tables Menu in your wordpress admin navigation.
3. Simply Add WP Tables and use shortcode in page or post.
